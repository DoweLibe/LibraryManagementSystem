# Library System (Console Version)

A Java console application for managing books and magazines. Includes searching, sorting, and checkout tracking using a 2D array.

## Features
- Add books and magazines
- Search by title
- Delete items
- Sort by title or year
- Track monthly checkouts (2D array)

## Skills Demonstrated
- Abstract classes
- Inheritance
- Interfaces
- Sorting and searching
- 2D arrays
